# SnapBox 📸✨  

SnapBox is a vibrant platform where you can share your memories, view others' posts, and stay connected. Built for ease and creativity, it allows users to upload, edit, and explore a gallery of shared experiences.  

---

## 🚀 Features  

🔐 **Secure Authentication**  
- User login and registration to ensure safe access.  

📤 **Post Memories**  
- Upload photos, add captions, and share your moments with others.  

👀 **Explore Posts**  
- View posts from other users, like, and engage with their content.  

✏️ **Edit or Update Posts**  
- Update your posts with new captions or replace images effortlessly.  

---

## 🛠️ Technology Stack  

- **Frontend:** React.js  
- **Backend:** Node.js, Express.js  
- **Database:** Appwrite  

---

## 📝 Usage  

- **Sign Up/Login:** Create an account or log in to access the platform.  
- **Post Memories:** Upload photos with captions to share your moments.  
- **Explore:** Browse and interact with posts shared by others.  
- **Edit Posts:** Update your existing posts with ease.  

---

## 📦 Installation  

Follow these steps to set up and run SnapBox locally:  

📥 **Clone the repository:**  
```bash
git clone https://github.com/AyushiMaithani/SnapBox.git
  ```

 📂 Change the working directory:
 ```bash
  cd SnapBox
  ```

🔧 Install dependencies:
```bash
  npm install
```

🛠️ Set up environment variables:

Create a .env file in the root directory and add the following:
```env
VITE_APPWRITE_URL=your-appwrite-url
VITE_APPWRITE_PROJECT_ID=your-project-id
VITE_APPWRITE_DATABASE_ID=your-database-id
VITE_APPWRITE_COLLECTION_ID=your-collection-id
VITE_APPWRITE_BUCKET_ID=your-bucket-id
VITE_RTE_API_KEY=your-rte-api-key
```

🚀 Start the application:
```bash
  npm run dev
```
